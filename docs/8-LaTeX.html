8 LaTeX

8.1 TeX Liveのインストール

macOS Monterey で TeX Live を使う方法。 Intelではなく，Apple siliconであることを想定している（一部異なる）。

以下のいずれかでインストールする。 インストールとTeXShopの初期設定は1の方が楽だが，2の方がインストール後に苦労しない。

困ったことがあっても基本的に誰も助けてくれない。 TeXユーザーはわからない人が何がわからないのかがわからない。 わからない人は何がわからないのかを説明できない。 すなわち，地獄。

知る限り，MacTeX をおすすめできる場合とそうでない場合（私見） が最も親切な説明。

以下の2つの方法の2番目がおすすめ。 自信がない場合は，1の方を強く推奨する。

  1. MacTeX のインストーラーを使って，TeXLive-2022のみをインストールする。Ghostscriptはインストールしない方がよい。

  2. Installing TeX Live over the Internet から， install-tl-unx.tar.gz をダウンロードして，インストールする。

$ tar xvzf install-tl-unx.tar.gz
$ cd ./install-tl-20220405
$ sudo ./install-tl --no-gui
$ sudo /usr/local/texlive/2022/bin/universal-darwin/tlmgr path add
$ sudo tlmgr update --self --all

念のため，PATHを確認しておく。

$ kpsewhich -var-value TEXMF
$ cat /usr/local/texlive/2022/texmf.cnf

8.2 Ghostscriptのインストール

Homebrewを使って，Ghostscriptをインストールする。 まず，Homebrewをインストールする。

$ cd /opt
$ curl -L https://github.com/Homebrew/brew/tarball/master | tar xz --strip 1 -C homebrew

Intelの場合は以下を実行する。

$ /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
$ # /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

PATHを通す。

$ export PATH=$PATH:/opt/homebrew/bin

次に，Ghostscriptをインストールする。

$ brew install ghostscript

無事にインストールできたかを確認したければ，

$ gs -version

または

$ which gs

を実行する。 それっぽいものが返ってくると，Ghostscriptのインストールに成功している。 エラーが出たり，何も返ってこなければインストールに失敗している。

MacTeX のGhostscriptをアンインストールした後に，Homebrewを使って，Ghostscriptをインストールする。

$ brew install ghostscript

ゴミが残っている場合，インストールが完了しない。 リンクで失敗する。 慎重になりたければ，以下を実行し，警告に従って，ファイルを削除するなどする。

$ brew link ghostscript

よくわからなければ以下を実行すればよい。

$ brew link --overwrite ghostscript

うまくいったかどうかは，以下のコマンドを実行して確認できる。

$ gs -version

8.4 ヒラギノを埋め込む

以下を実行する前に，TeX LiveとGhostscriptをインストールしておく必要がある。

$ kanji-config-updmap-sys status # フォントを確認
$ sudo tlmgr repository add http://contrib.texlive.info/current tlcontrib
$ sudo tlmgr pinning add tlcontrib '*'
$ sudo tlmgr install japanese-otf-nonfree japanese-otf-uptex-nonfree ptex-fontmaps-macos cjk-gs-integrate-macos
$ sudo cjk-gs-integrate --link-texmf --cleanup
$ curl https://raw.githubusercontent.com/texjporg/cjk-gs-support/master/cjk-gs-integrate-macos.pl -o cjk-gs-integrate-macos.pl
$ sudo perl cjk-gs-integrate-macos.pl --link-texmf
$ sudo kanji-config-updmap-sys --jis2004 hiragino-highsierra-pron
$ kanji-config-updmap-sys status # hiragino-highsierra-pron になった
$ # sudo kanji-config-updmap-sys auto # morisawa になった

なぜヒラギノを埋め込むのかは聞いてはいけない。

最近は，sudo mktexlsr は必要ないっぽい（「っぽい」の正確性を確認しようとすることは地獄への誘い。やめておく）。

8.5 TeXShopの設定

「設定プロファイル」から「upTeX (ptex2pdf)」を選択する。

$ defaults write TeXShop FixLineNumberScroll NO
$ defaults write TeXShop SourceScrollElasticity NO
$ defaults write TeXShop FixPreviewBlur YES
