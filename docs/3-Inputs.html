3.2 文字入力

3.2.1 en dash (–)とem dash (—)の入力方法

macOS

  • en dash: ⌥ + -
  • em dash: ⌥ + ⇧ + -

Windows

  • en dash: Alt + 0150 or Ctrl + テンキーの-
  • em dash: Alt + 0151 or Ctrl + Alt + テンキーの-

3.3 文字化け

メールの文字化けは以下で対応する。

  • 文字化けしたメールの修復

添付ファイルにwinmail.datが付いている場合，その中身を確認するためには，そのファイルを変換しなければならない。

  • winmail.dat変換ツール(インストール不要)

ただし，ファイルサイズが大きすぎる場合，変換できないため，Google検索して，適当なサービスを利用する。 一方，ファイルサイズが小さすぎる場合，ゴミであることがある。

ウェブブラウザの文字化けには，テキストエンコードを変更する必要がある。 どのエンコードにすべきかわからなければ，日本語の中から順番に変えていけばよい。
